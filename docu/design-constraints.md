# Design Constraints & UI Patterns

## Purpose

This file defines the most important design constraints and UI patterns for this project.

The goal is to ensure the website looks:

- clean
- sharp
- premium
- modern
- trustworthy
- restrained
- client-ready

These rules must always be followed.

Do not design this website like a generic template.
Do not overdesign it.
Do not add unnecessary visual noise.

Design with restraint.

Prioritize:

- spacing
- typography
- hierarchy
- alignment
- clean composition
- conversion clarity

over decorative effects.

---

## Overall Design Principle

The website should feel like a **premium Swiss chauffeur service**.

Visual impression:

- calm confidence
- refined luxury
- elegant professionalism
- modern trustworthiness

This is **not**:

- a flashy limousine party website
- a cheap taxi website
- an overanimated landing page
- a generic AI template

The final design must feel polished and intentional.

---

## Core Constraints

### 1. Spacing System

Use a consistent spacing scale only.

Allowed spacing values:

- 4px
- 8px
- 16px
- 24px
- 32px
- 48px
- 64px
- 96px

Do not use random spacing values.

Spacing must feel deliberate and rhythmic.

Large sections should have generous vertical spacing.

---

### 2. Content Width

Use a centered layout with controlled content width.

Rules:

- standard content max-width: 1200px
- larger visual sections may go to 1280px
- text content blocks should remain narrower for readability
- always center the main layout

Do not allow text or cards to stretch too wide.

---

### 3. Border Radius

Use only two border radius values.

Allowed values:

- small radius: 8px
- large radius: 16px

Do not mix many different corner styles.

---

### 4. Shadow System

Use only two shadow levels.

Suggested usage:

- soft shadow for standard cards
- elevated shadow for featured cards or hover states

Shadows must be subtle and premium.

Avoid:

- dark heavy shadows
- blurry oversized shadows
- stacked shadow effects

---

### 5. Color Discipline

Use a restrained luxury color palette.

Allowed color roles:

- background
- surface
- dark text
- muted text
- border
- premium accent

Suggested direction:

- warm white / ivory background
- charcoal / graphite text
- muted champagne / warm gold / beige accent
- very soft neutral borders

Avoid:

- bright yellow gold
- neon tones
- strong blue accents
- overly saturated colors
- too many different accent colors

Use very few colors and use them consistently.

---

### 6. Typography Hierarchy

Typography must carry the premium feel.

Use a clean and restrained hierarchy.

Recommended scale:

- H1: 48px
- H2: 36px
- H3: 24px
- Body: 16px
- Small text: 14px

Recommended line heights:

- headings: 1.15–1.25
- body text: 1.5–1.7

Rules:

- serif for major headings
- sans-serif for body text
- avoid overly long paragraphs
- use short readable blocks
- maintain clear heading rhythm

Typography should feel elegant, not loud.

---

### 7. Grid & Alignment

Use consistent layout alignment.

Rules:

- use grid or flex layouts deliberately
- align cards cleanly
- maintain consistent gaps
- sections should follow a stable rhythm
- do not place elements arbitrarily

The layout should feel structured and composed.

---

## Visual Restraint Rules

Always design with restraint.

Avoid:

- too many gradients
- unnecessary glassmorphism
- loud luxury clichés
- big glowing effects
- too many decorative lines
- excessive badges
- crowded card layouts
- multiple competing focal points

The website should feel premium because of:

- proportion
- hierarchy
- spacing
- materials
- motion quality

not because of visual excess.

---

## Section Design Patterns

## 1. Hero Pattern

The hero must be clear and disciplined.

Hero should contain only:

- one strong headline
- one supporting paragraph
- two CTA buttons
- three trust indicators

Optional:
- premium background image
- subtle overlay

Do not overload the hero.

The hero must communicate within seconds:

- what the company does
- why it feels premium
- how to contact immediately

### Hero design rules

- large clean heading area
- strong CTA visibility
- enough whitespace
- premium but calm visual tone
- high contrast and readability

---

## 2. CTA Pattern

CTA buttons are central to conversion.

CTA design must feel:

- premium
- tactile
- clear
- elegant

### CTA rules

- one clearly dominant primary CTA
- one refined secondary CTA
- use concise labels
- make buttons easy to tap on mobile
- repeat CTAs naturally throughout the page

Avoid:

- too many button styles
- overly playful rounded buttons
- aggressive colors
- oversized button text

Use consistent button sizing and spacing.

---

## 3. Card Pattern

Use one card design language across the site.

Cards should feel:

- clean
- elevated
- premium
- structured

Card pattern should include:

- soft border
- subtle shadow
- consistent padding
- clean spacing
- strong heading hierarchy
- optional muted supporting text

Avoid clutter inside cards.

Every card must have a clear focal point.

---

## 4. Vehicle Showcase Pattern

The vehicle section is a key visual highlight.

Treat vehicle cards like premium product cards.

They should feel more elevated than the rest of the cards, but still restrained.

### Vehicle card design ideas

- larger card size
- premium image framing
- subtle layered background
- elegant accent line or badge
- soft highlight
- refined hover lift
- clear content grouping
- strong typography

### Vehicle card content structure

- vehicle name
- short luxury description
- 2–3 concise highlights
- CTA button

### Vehicle design rules

- visually special, but not flashy
- premium and polished
- no aggressive car marketplace style
- no dark overdesigned “luxury” clichés
- images should feel clean and editorial

Think:
executive brochure aesthetic + modern landing page clarity

---

## 5. Popular Routes Pattern

This section must feel practical and conversion-friendly.

The user should immediately recognize a useful route and click.

### Route card rules

- easy to scan
- route title first
- muted description second
- distance visible but secondary
- clear CTA
- optional secondary call action

Do not design this like a pricing table.

It should feel like a shortcut to contact.

---

## 6. Driver Introduction Pattern

The driver section should humanize the service and build trust.

It must feel:

- professional
- personal
- reliable
- calm

### Driver section rules

- use a clean two-column layout
- present photo respectfully
- keep text concise
- add trust highlights
- include CTA beneath or beside text

Avoid:

- overly emotional storytelling
- too much biography
- casual language
- awkward portrait styling

This should feel like a premium founder/service introduction.

---

## 7. Trust Section Pattern

The “Why Choose Us” area should be very readable.

Use either:

- icon cards
- a clean checklist grid
- a simple premium feature list

Trust points should be concise.

Do not write long marketing paragraphs here.

Focus on:

- punctuality
- comfort
- reliability
- premium vehicles
- chauffeur professionalism
- 24/7 availability

---

## Motion & Interaction Constraints

Use motion sparingly.

Allowed motion types:

- subtle fade-up on scroll
- gentle card lift on hover
- elegant button hover transition
- smooth anchor scrolling
- calm mobile menu animation

Avoid:

- bounce effects
- dramatic zoom
- rotating elements
- long chained animations
- excessive parallax
- distracting movement

Motion should improve polish, not draw attention to itself.

---

## Mobile Design Constraints

Mobile experience must remain premium.

Rules:

- strong CTA visibility above the fold
- clear spacing on small screens
- comfortable tap targets
- floating call CTA on mobile
- stacked layouts should still feel intentional
- cards must remain elegant on small screens
- no cramped text blocks

Do not let mobile become a generic collapsed layout.

---

## Sharpness & Cleanliness Rules

To make the UI feel sharp:

- use subtle borders to define surfaces
- use consistent card padding
- align content precisely
- avoid inconsistent widths
- keep text blocks compact
- use whitespace to create structure
- maintain clean visual rhythm between sections

Premium sharpness comes from discipline, not decoration.

---

## What Good Looks Like

The final design should feel similar in quality to modern premium landing pages that emphasize:

- clarity
- spacing
- typography
- product-like presentation
- elegant restraint

The user should feel:

- this business is trustworthy
- this service is premium
- this website is clean and professional
- contacting them is easy

---

## Final Design Check

Before finalizing, check the design against these questions:

1. Does the page feel premium without trying too hard?
2. Is the layout clean and spacious?
3. Are the CTAs immediately visible and clear?
4. Does the vehicle section feel special and polished?
5. Does the mobile version still feel refined?
6. Is the design restrained enough to feel credible?
7. Would this look believable for a premium chauffeur service in Switzerland?

If the answer is no, refine the design.

---

## Priority Order

When design decisions conflict, prioritize in this order:

1. clarity
2. spacing
3. typography
4. alignment
5. conversion
6. restraint
7. decorative styling

Never sacrifice clarity for decoration.
